Privacy Policy for Offby1 Chess

Last updated: January 13, 2026

Introduction

Offby1 Chess ("we", "our", or "the app") is committed to protecting your privacy. This Privacy Policy explains how information is handled when you use our Android chess application.

Information Collection and Use

Offby1 Chess does not collect, store, or transmit any personal information about you.

The app operates by connecting to a chess server where game data is stored. However:

  • No user accounts or registration are required
  • No personally identifiable information is collected
  • No email addresses, names, or contact information are requested
  • No usage analytics or tracking data is collected
  • No advertising identifiers are used

Permissions Used

The app requests the following Android permissions:

  • INTERNET: Required to connect to chess servers and synchronize game data
  • ACCESS_NETWORK_STATE: Used to check network connectivity before making API calls to provide a better user experience

These permissions are used solely for the app's core functionality of playing chess games and are not used for any data collection purposes.

Game Data

When you play games in Offby1 Chess, the following data is sent to the configured chess server:

  • Chess moves (in UCI notation, e.g., "e2e4")
  • Game state (board position, whose turn, etc.)
  • AI difficulty setting

This data is associated with a randomly generated game ID (UUID) and is not linked to you personally. Games are accessible only by knowing the game ID.

Server Configuration

The app allows you to configure custom chess servers. By default, it connects to https://chess.offby1.info. When you play games, move data is sent to your configured server.

Server URL preferences are stored locally on your device and are never transmitted to us or any third party.

Local Data Storage

The app stores the following data locally on your device:

  • Server URL configuration
  • App preferences and settings
  • Cached game data for offline viewing

This data remains on your device and is deleted if you uninstall the app.

Third-Party Services

The app does not use any third-party analytics, advertising, or tracking services. The only network communication is with the chess server you configure.

Children's Privacy

The app does not knowingly collect any information from children. No registration, personal information, or accounts are required to use the app, making it suitable for users of all ages.

Data Security

Since the app does not collect personal information, there is no personal data to secure. Game data transmitted to servers is sent over HTTPS (encrypted connections) when available.

Your Rights

Since no personal data is collected, there is no personal information to access, modify, or delete. Game data stored on servers can be deleted by removing games within the app.

Changes to This Privacy Policy

We may update this Privacy Policy from time to time. Any changes will be posted at this URL, and the "Last updated" date will be revised. Continued use of the app after changes indicates acceptance of the updated policy.
